# たまごっち型ライフログアプリ 仕様書

今すぐプレイ
https://mayo135.github.io/eggHackthon/

## 1. 概要

大学生活の活動を記録し、その内容に応じてキャラクター（卵）が成長するライフログWebアプリ。
活動カテゴリの蓄積に応じて4パターンに進化分岐する。

## 2. 利用者

- 一般大学生

## 3. システム目的

- 大学生活の活動履歴を蓄積・可視化する
- 学生の成長過程をゲーム的に表現し、モチベーションを向上させる
- 就職活動時のガクチカ整理を支援する

## 4. 技術構成

- HTML / CSS / JS のみ（サーバー不要）
- データ保存：localStorage
- 1ページ構成（タブ切り替え）

## 5. ファイル構成と担当分担

| ファイル | 役割 | 担当 |
|---------|------|------|
| `index.html` | 画面構造 + タブ | 担当A（main.jsと兼任） |
| `style.css` | スタイリング | 担当A or デザイン担当 |
| `main.js` | タブ切り替え・初期化・学年進行・キャラ表示 | 担当A |
| `activity.js` | イベントデータ定義・活動記録・localStorage読み書き | 担当B |
| `growth.js` | 成長判定ロジック・進化ルート決定 | 担当C |

## 6. 画面構成（3タブ）

### 6.1 メインタブ

- 現在の学年表示（1年〜4年）
- キャラクター画像
- 「学年を進める」ボタン → 成長判定を実行し、キャラが進化

### 6.2 活動記録タブ

- イベント選択（プルダウン）
- 活動日入力
- メモ（任意）
- 「記録する」ボタン

### 6.3 履歴タブ

- 記録された活動の一覧表示

## 7. 活動カテゴリと進化パターン

### 7.1 4カテゴリ

| カテゴリ | 進化先の例 |
|---------|-----------|
| 運動系 | 体育会系キャラ |
| 知識系 | 研究者キャラ |
| 技術系 | エンジニアキャラ |
| ビジネス系 | ビジネスキャラ |

### 7.2 イベントと重み付け

各イベントはカテゴリへの比率（合計1.0）と強度（power）を持つ。

```js
const EVENTS = [
  { name: "ハッカソン",       weights: { sports: 0, knowledge: 0.4, tech: 0.6, business: 0 }, power: 3 },
  { name: "体育大会",        weights: { sports: 0.9, knowledge: 0, tech: 0, business: 0.1 }, power: 2 },
  { name: "学会発表",        weights: { sports: 0, knowledge: 0.8, tech: 0.2, business: 0 }, power: 3 },
  { name: "バイト",          weights: { sports: 0, knowledge: 0, tech: 0, business: 1.0 }, power: 1 },
  { name: "技術系インターン",  weights: { sports: 0, knowledge: 0.1, tech: 0.7, business: 0.2 }, power: 3 },
  { name: "就労型インターン",  weights: { sports: 0, knowledge: 0.1, tech: 0.1, business: 0.8 }, power: 3 },
  // ... 追加可能
];
```

活動を記録するたびに `power × weights` が各カテゴリのスコアに加算される。

## 8. 成長判定ロジック

```
「学年を進める」押下時：
  1. 現在の各カテゴリスコアを取得
  2. 最もスコアの高いカテゴリを判定
  3. そのカテゴリに対応するキャラ画像に切り替え
  4. 学年を +1（最大4年）
```

拡張ポイント：将来的にスコア閾値やカテゴリ比率による細かい分岐を追加可能。

## 9. キャラクター画像

|------|------|

※後から3年用・4年用を別画像に分離可能。

## 10. データ構造（localStorage）

```js
{
  year: 1,                    // 現在の学年
  scores: { sports: 0, knowledge: 0, tech: 0, business: 0 },
  activities: [               // 活動履歴
    { event: "ハッカソン", date: "2026-03-20", memo: "..." }
  ],
  currentType: null           // 進化タイプ（null=卵）
}
```
